# ai-business-stack

Opinionated, agentic AI stacks for bootstrapping a startup from zero to paying users.

Each stack under [stacks/](stacks/) is a self-contained recommendation for a specific kind of business. Stacks are split into **substacks** (Tech, Marketing, ...) that can be built and evaluated independently.

## Stacks

| Stack | Best for | Status |
|---|---|---|
| [brand-new-startup](stacks/brand-new-startup/) | Self-serve businesses (SaaS, PLG, DTC, creator) where checkout closes the sale | Active |

## Principles

- **Opinionated.** One primary pick per job. Alternatives appear only when the tradeoff is real.
- **Agentic-first.** Tools that expose agent APIs, MCP servers, or slot into an AI-driven workflow are preferred over pretty UIs.
- **Bootstrapper economics.** Free tiers and usage-based pricing preferred. Paid tools only when they replace a person.
- **Time-stamped.** AI tooling churns monthly. Every substack README carries a `Last updated` date — if it's stale, treat it as a starting point, not gospel.
- **Independently usable.** You can adopt any substack without the others.

See [stacks/README.md](stacks/README.md) for how stacks are organized and how to add a new one.
